Photon as Arduino

Flash to the Photon as an .ino file





Using particle.io load the console and see if you can read the values sent live to the web.
The push button should control the red LED. The photoresistor should control the Yellow LED strength. See if you can make the analog side work in reverse?
Hint: the voltage devider used for the photoresistor can be reversed ground and 3V3.
